
## Weather Web App using PHP 
## Usage <br>
Setup the repository <br>
```
git clone https://github.com/coswat/weather-app.git
cd weather-app

```
## Api

Add Api in `Weather.php` in Line `29`

```php

//you will get Api from https://openweathermap.org
$apikey = 'API KEY HERE';

```
